# SHIELD
Synchronized Hazard Information & Emergency Live Dashboard (SHIELD)

## Installation Guide:
Open VS Code, open a terminal.

1) Clone the repository:
```bash
git clone https://github.com/LawrenceTanny/SHIELD
```

2) Then install dependencies and libraries:
```bash
npm install
```

3) To be safe, and have the depedencies fixed(OPTIONAL):
```bash
npm audit fix --force
```

4) Run the development server:
```bash
npm run dev
```
